NHTSA Vehicle Recalls Analysis

Presentation Notes

This project is a comprehensive 25-year analysis (2000-2025) of vehicle recall data in United States, sourced from Data.gov using a combination of SQL for querying and Microsoft Excel for data visualization, I meticulously cleaned and procesed thousands of records to uncover key patterns and actionable insights. This goal of this analysis is to identify trends in vehicle recalls across manufacturers, vehicle types, and defect categories over the past two and a half decades. Through advanced query techniques, I extracted deep insights from raw data, which are presented through an interractive dashboard that allow users to explore recall trends year by year, and also a PDF presentation of this analysis. Whether you are a customer, researcher, or auomobile saftey advocate, this tool aims to bring clarity to historical recall data and support informed decision-making in the automotive space.
